# KM Engenharia & Tecnologia

## Sobre o Projeto

Site completo e responsivo para empresa de Engenharia Civil e Ambiental, desenvolvido com HTML5, CSS3 e JavaScript puro.

## Características

- **Design Moderno**: Interface limpa e profissional com foco em sustentabilidade
- **Totalmente Responsivo**: Adaptável a todos os dispositivos (desktop, tablet, mobile)
- **Performance**: Código otimizado e sem dependências externas pesadas
- **SEO Friendly**: Estrutura semântica e meta tags otimizadas
- **Acessibilidade**: Navegação por teclado e leitores de tela

## Funcionalidades

### Seções Principais
- **Hero**: Banner inicial com estatísticas animadas
- **Serviços**: 6 cards de serviços com animações hover
- **Diferenciais**: Grid de diferenciais da empresa
- **Projetos**: Galeria filtrável de portfólio
- **Sustentabilidade**: Calculadora de impacto ambiental
- **Depoimentos**: Slider automático de clientes
- **Blog**: Grid de artigos técnicos
- **Contato**: Formulário completo com mapa

### Recursos Interativos
- Navegação suave (smooth scroll)
- Menu mobile hambúrguer
- Filtro de projetos por categoria
- Calculadora de sustentabilidade
- Chat widget flutuante
- Botão voltar ao topo
- Animações de entrada (fade in)
- Contador de estatísticas animado

## Como Usar

1. Faça o download dos arquivos
2. Extraia para uma pasta
3. Abra o arquivo `index.html` em um navegador
4. Para publicar, faça upload para seu servidor web

## Personalização

### Cores
Edite as variáveis CSS no início do arquivo `css/style.css`:

```css
:root {
    --primary-green: #2E7D32;
    --accent-orange: #FF6F00;
    /* ... outras variáveis */
}
```

### Conteúdo
- Substitua as imagens na pasta `images/`
- Atualize textos diretamente no `index.html`
- Modifique informações de contato na seção correspondente

### Funcionalidades
- Ajuste a calculadora no arquivo `js/main.js`
- Configure o formulário de contato com seu backend
- Personalize mensagens do chat widget
